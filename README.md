# 🍽️ RN-Food

React Native로 개발한 맛집 추천 애플리케이션입니다.

## 📱 프로젝트 소개

사용자의 위치와 취향을 기반으로 주변 맛집을 추천하고 리뷰를 공유할 수 있는 모바일 앱입니다.

## 🛠️ 기술 스택

- **React Native** - 크로스 플랫폼 모바일 앱 개발
- **Expo** - React Native 개발 환경
- **NativeWind** - Tailwind CSS 기반 스타일링

## 🚀 시작하기

### 설치

```bash
npm install
```

### 실행

```bash
npx expo start
```

실행 후 다음 옵션으로 앱을 열 수 있습니다:

- `i` - iOS 시뮬레이터
- `a` - Android 에뮬레이터
- QR 코드 스캔 - Expo Go 앱

## 📂 프로젝트 구조

```
RN-Food/
```
